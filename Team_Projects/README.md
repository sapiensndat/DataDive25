# Projects

Projects folder for individual team submissions.

## Instructions

Create a copy of the template SampleTeam folder and name it after your team! 

So replace SampleTeam with your team name.  For example, if your team name is "Job Insights", then your folder should be named "JobInsights" or "job_insights"



Please fill out the project.md file with your team name, members, challenge area, and project description.


# NOTE: 

1. If you use any sensative data or API keys in your project, please do not commit them to this repository. 
2. Do not commit datasets or files that are larger than 50MB to this repository.

You are welcome to include powerpoints or pdfs and smaller datasets alongside your project.md readme file.
